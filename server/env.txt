----------Deploy:----------

PORT= -> dejar que la pag defina
DB_DEPLOY= -> definir según db desplegada
URL_CLIENT= -> definir según url del cliente
API_KEY= -> definir según api de preferencia

iniciar con "npm run start-server"

----------Desarrollo:------

PORT= -> en caso de no definir queda en 3001
DB_USER= ej:postgres
DB_PASSWORD= ej:password
DB_HOST= ej:localhost:1234
URL_CLIENT= -> en caso de no definir queda en localhost:3000
API_KEY= -> definir según api de preferencia

iniciar con "npm start"
